# Claude Chatbot Demo v1

A Streamlit application that provides a chat interface with Anthropic's Claude model using LangChain.

## Features

- Chat interface powered by Claude 3 Sonnet
- Customizable system prompt
- Secure API key input
- Conversation history maintained in session state

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Get an Anthropic API key from [console.anthropic.com](https://console.anthropic.com/)

3. Run the application:
```bash
streamlit run app.py
```

4. Enter your Anthropic API key and customize the system prompt in the sidebar

5. Start chatting with Claude!
